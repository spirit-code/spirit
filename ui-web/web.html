About Spirit

Spin Simulation Framework

Spirit version x


Spirit is platform-independent, open-source code with optional visualization, written in C++11. The source code is available on GitHub under the terms of the MIT license.

While this web-based demo version is limited to Spin Dynamics simulations, the Spirit code has been developed as a flexible solution to various use-cases, including:

  • Spin Dynamics simulations obeying the Landau-Lifschitz-Gilbert equation
  • Direct Energy minimisation of a spin system
  • Minimum Energy Path calculations for transitions between different spin configurations, using the GNEB method

The visualisation is performed by VFRendering.

More details may be found on GitHub or in the documentation.

Main author of the Spirit code is Gideon Müller (Quantum Theory of Materials department of the Peter Grünberg Institute ).

The Web Interface is being developed by Florian Rhiem (Scientific IT-Systems, PGI/JCNS Technical Services and Administration).
